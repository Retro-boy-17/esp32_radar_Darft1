# ESP32 Radar System
Ultrasonic radar using ESP32, HC-SR04, SG90 servo and Processing visualizer.

## Components
- ESP32
- HC-SR04 ultrasonic sensor
- SG90 servo motor
- Resistors for voltage divider

## How to run
1. Upload radar_esp32.ino to ESP32
2. Open radar_visualizer.pde in Processing
3. Run Processing sketch
